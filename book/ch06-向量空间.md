# 第六章：向量空间——什么才是"空间"

> 向量"住"在哪里？什么样的集合配得上被叫做"空间"？

---

## 函数也是向量？

上一章结尾留了一个问题：向量住在哪里？

在回答这个问题之前，让我先讲一件更让人困惑的事。

十九世纪初，法国数学家傅里叶在研究热传导问题的时候，做了一件惊人的事：他说任何一个"合理"的函数都可以拆成一堆正弦函数和余弦函数的加权求和。比如一个锯齿形的波，看着跟光滑的正弦波八竿子打不着，但傅里叶说：没事，你给我足够多的正弦和余弦，我就能把它拼出来。

$$f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n \cos nx + b_n \sin nx \right)$$

这个公式的细节不重要，重要的是它在做的事情——拿一堆函数，每个乘以一个系数（那些 $a_n$ 和 $b_n$），然后加起来，拼出一个新函数。

等等。这个操作听着耳熟吗？

"拿一堆东西，每个乘以一个系数，然后加起来"——这不就是上一章讲的**线性组合**吗？当时我们说的是拿几个向量（箭头），每个乘以一个数，加起来得到新向量。现在傅里叶拿的是函数，每个乘以一个系数，加起来得到新函数。操作的"形状"一模一样，只是参与者从箭头变成了函数。

这就怪了。函数是"向量"吗？函数没有方向，没有长度（至少不是通常意义上的长度），没有箭头——它怎么能是向量呢？

再想一步。多项式也可以做线性组合：

$$3(x^2) + 5(x) + (-2)(1) = 3x^2 + 5x - 2$$

三个多项式 $x^2$、$x$ 和 $1$，分别乘以系数 $3$、$5$、$-2$，加起来得到一个新多项式。还是线性组合。

如果连函数和多项式都能做线性组合，那"向量"到底是什么？难道"有大小有方向的箭头"这个定义太窄了？

答案是：确实太窄了。箭头只是向量的一种具体形态——就像"人"不只是"你认识的那几个人"。要理解向量的真正含义，你需要一个更高层次的定义。

这个定义的核心思想不是"向量是什么东西"，而是"向量能做什么操作"。

## 皮亚诺的"游戏规则"

1888 年，意大利数学家朱塞佩·皮亚诺出版了一本叫《几何计算》（*Calcolo geometrico*）的书。

皮亚诺这个人，你可能在别的场合听过他的名字——他最出名的是给自然数做了公理化定义（就是那个"皮亚诺公理"），搞清楚了 $1, 2, 3, 4, \ldots$ 到底是什么。不过在这本《几何计算》里，他做了另一件同样重要的事：他给出了接近现代形式的向量空间定义，以及线性映射的定义。

这里要加一句公平的话：皮亚诺不是凭空冒出来的。在他之前，格拉斯曼（上一章那位写了没人看的天书的中学教师）已经勾勒了任意维度线性结构的基本轮廓，法国数学家达布等人也做过一些比较局部的公理化尝试。但皮亚诺的工作被公认为最接近"现代向量空间定义"的第一个版本——他把前人的直觉整理成了清晰的公理体系。

皮亚诺的核心思路可以用一句话概括：**不关心向量"是"什么，只关心向量"能做"什么。**

这种思路在数学里有个名字，叫"公理化方法"。它的精神跟法律有点像——法律不关心你是谁（你的身高、长相、星座、血型统统不管），只关心你做了什么（你有没有遵守交通规则、有没有按时纳税）。只要你的行为符合法律的所有规定，你就是一个守法公民，不管你是人还是外星人。

皮亚诺对向量做了同样的事：不管你是箭头、函数、多项式还是别的什么东西，只要你的行为满足一组特定的规则，你就是"向量"，你所在的集合就是"向量空间"。

那么这些规则是什么呢？

## 入会须知

想象你面前有一个俱乐部，叫"向量空间俱乐部"。任何集合都可以申请加入，但必须满足全部入会条件。门口挂着一张告示：

**基本能力要求：**

你的成员之间必须能做两件事——**加法**（任意两个成员可以相加，结果还是你的成员）和**数乘**（任何一个成员都可以乘以一个数，结果还是你的成员）。

用数学的话说，加法和数乘必须满足"封闭性"——做完运算不会跑到集合外面去。这就好比说，如果你开了一家只收现金的商店，那顾客之间不管怎么交易，钱还是在顾客群体里流通，不会突然变成比特币跑掉。

**加法必须遵守的规矩：**

1. **交换律**：$\mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}$。你加我和我加你一样。没有先来后到之分。
2. **结合律**：$(\mathbf{u} + \mathbf{v}) + \mathbf{w} = \mathbf{u} + (\mathbf{v} + \mathbf{w})$。三个人凑份子，先把前两个人的钱加一起再加第三个人的，和先加后两个人的钱再加第一个人的，结果一样。
3. **零元素**：必须存在一个特殊的成员 $\mathbf{0}$，任何成员加上它都不变：$\mathbf{v} + \mathbf{0} = \mathbf{v}$。这就是那个"什么都不做"的成员——加了等于没加。
4. **逆元素**：每个成员 $\mathbf{v}$ 都有一个"反面" $-\mathbf{v}$，两个加起来等于零：$\mathbf{v} + (-\mathbf{v}) = \mathbf{0}$。你做了一步，总能原路退回。

**数乘必须遵守的规矩：**

5. **$1$ 乘以谁都是谁**：$1 \cdot \mathbf{v} = \mathbf{v}$。用 $1$ 去缩放，等于什么都没干。
6. **数乘的结合律**：$a(b\mathbf{v}) = (ab)\mathbf{v}$。先用 $b$ 缩放再用 $a$ 缩放，等于直接用 $ab$ 缩放。你把照片先放大两倍再放大三倍，跟直接放大六倍是一回事。
7. **数乘对向量加法的分配律**：$a(\mathbf{u} + \mathbf{v}) = a\mathbf{u} + a\mathbf{v}$。把两个东西加起来再统一缩放，等于分别缩放再加起来。
8. **数乘对数字加法的分配律**：$(a + b)\mathbf{v} = a\mathbf{v} + b\mathbf{v}$。用两个倍数之和去缩放，等于分别缩放再加起来。

八条规矩，一条都不能少。

你可能会说：这也太多了吧？而且好多条看起来就是"废话"啊——交换律、结合律、分配律，这不是天经地义的吗？

还真不是。记得上一章的四元数吗？四元数的乘法就不满足交换律。还有很多看似自然的运算体系，仔细一检查会发现缺这条少那条。这八条公理之所以每一条都要写出来，是因为它们各自独立——你不能从其中几条推导出剩下的。少一条，就有可能放进来一些"不该进来"的东西，导致后面的定理全部崩塌。

数学家的"啰嗦"不是没有道理的。这就像法律条文——外行人看着觉得废话连篇，但每一条都是某次惨痛教训的总结。你以为这条规定多余？那是因为你还没遇到有人钻这个空子。

## 验货时间

好了，现在让我们回头看看，那些让我们困惑的东西——函数、多项式——真的能通过"验货"吗？

**箭头向量（$\mathbb{R}^n$）**：二维箭头、三维箭头——这是最经典的例子。两个箭头可以头尾相接做加法，箭头可以拉长缩短做数乘。八条公理全部满足。恭喜你，$\mathbb{R}^2$ 和 $\mathbb{R}^3$ 是向量空间，没有悬念。

**多项式**：所有次数不超过 $n$ 的多项式构成的集合。两个多项式可以相加——$x^2 + 2x$ 加上 $3x^2 - x$ 等于 $4x^2 + x$，结果还是次数不超过 $n$ 的多项式。多项式可以乘以一个数——$5(x^2 + 2x) = 5x^2 + 10x$，还是多项式。零多项式（所有系数都是零）就是那个零元素。交换律、结合律、分配律……全部满足。通过！多项式的集合是向量空间。

**函数**：所有从实数到实数的连续函数构成的集合。两个函数可以相加——$(f+g)(x) = f(x) + g(x)$；函数可以乘以一个数——$(cf)(x) = c \cdot f(x)$。零函数（处处等于零的函数）是零元素。八条公理？全部满足。通过！连续函数的集合也是向量空间。

你看到发生了什么吗？

箭头、多项式、函数——它们长得完全不一样，但它们的"行为模式"是一样的：都能做加法，都能做数乘，而且这些运算都遵守同一组规矩。公理化的魔力就在这里：它让你忽略表面的差异，直接抓住深层的共性。一个关于向量空间的定理，自动对箭头、多项式、函数全部成立——你不需要分别证明三次。

这就是抽象的力量。它不是远离现实的逃避，而是一种效率极高的思维压缩技术。

在抽象层面上，向量空间就是一个满足那八条公理的集合。它不关心你是箭头还是函数，只要你满足规则，你就是向量空间的一员。

普遍上讲，向量是一个n * 1的矩阵，这个可能更好理解一些，函数和多项式也可以看成是一个n * 1的矩阵，矩阵的每个元素就是函数在某个点的值或者多项式的某个系数。如果还是感觉理解不了，那就当向量是一个n * 1的矩阵。

## 空间里的"小空间"

有了向量空间之后，第一个自然的问题是：一个空间里面，能不能找到更小的空间？

当然可以。

三维空间 $\mathbb{R}^3$ 里，所有形如 $(x, y, 0)$ 的向量——也就是 $z$ 坐标为零的向量——构成一个二维平面。这个平面本身也是一个向量空间：平面内的向量相加还在平面内，平面内的向量乘以一个数还在平面内，八条公理全部满足。

这样的"大空间里面的小空间"，叫做**子空间**。

子空间有一个重要的性质：它必须包含零向量。换句话说，子空间必须"过原点"。一个不过原点的平面（比如三维空间里 $z = 1$ 这个平面）不是子空间——你拿里面的向量 $(1, 0, 1)$ 乘以零，得到零向量 $(0, 0, 0)$，但这个零向量不在 $z = 1$ 的平面上。封闭性被破坏了。

还有一个有趣的极端情况：$\{0\}$——只包含零向量的集合——也是子空间。它满足所有公理，虽然无聊透了，但规则就是规则。就像一个只有创始人一个人的俱乐部，它确实满足所有入会条件，只是没什么意思。

## 基：空间的"坐标轴"

向量空间有大有小。你怎么衡量一个空间的"大小"？

这就涉及到线性代数里最核心的概念之一：**基**。

让我先讲个比喻。你在一座陌生的城市，手里有一张地图。地图上标了经度和纬度——两个坐标。有了这两个坐标，城市里任何一个位置都可以被精确定位。这两个坐标方向（东西方向和南北方向）就是一组"基"——用它们的组合可以到达平面上的任何一个点。

而且，这两个方向缺一不可。只有东西方向不行——你只能沿着一条线移动。只有南北方向也不行。必须两个方向都有，你才能在整个平面上自由活动。

同时，两个方向就够了——你不需要第三个"东北方向"，因为东北方向可以由东和北组合出来。加第三个方向不会让你到达任何新的地方，只是增加了冗余。

这两个条件——"缺一不可"加上"足够覆盖整个空间"——就是基的定义。

正式一点说：一组向量如果满足两个条件，就是空间的一组**基**：

1. **线性无关**——没有任何一个向量可以由其余的线性组合得到。也就是说，每个向量都提供了不可替代的"新方向"。
2. **张成整个空间**——这组向量的所有线性组合可以覆盖空间中的每一个向量。用这几个向量当"原料"，通过缩放和加法，可以"调配"出空间中任何你想要的向量。

在 $\mathbb{R}^3$ 里，最自然的一组基是：

$$\mathbf{e}_1 = (1, 0, 0), \quad \mathbf{e}_2 = (0, 1, 0), \quad \mathbf{e}_3 = (0, 0, 1)$$

就是三根坐标轴方向的单位向量。任何三维向量 $(a, b, c)$ 都可以写成：

$$a \cdot \mathbf{e}_1 + b \cdot \mathbf{e}_2 + c \cdot \mathbf{e}_3$$

三个基向量，线性无关（互相垂直，谁也不能由别人组合出来），张成整个三维空间。

但这里有个关键的点——基不是唯一的。$\mathbb{R}^3$ 有无穷多组基。你可以选任意三个线性无关的向量作为基，比如 $(1, 1, 0)$、$(0, 1, 1)$、$(1, 0, 1)$ 也是一组基。就像描述一座城市的位置，你可以用经纬度，也可以用"距市中心的距离"加上"与正北方向的夹角"——不同的坐标系，同一座城市。

基不唯一，但有一个东西是不变的。

---

## 维数：独立方向的个数

不管你怎么选基，基里面的向量个数永远相同。

$\mathbb{R}^3$ 的基永远有三个向量。你可以换来换去，但不可能找到一组只有两个向量的基（两个向量最多张成一个平面，覆盖不了整个三维空间），也不可能找到一组四个向量的基（四个三维向量一定有冗余，不可能全部线性无关）。

这个"基里面向量的个数"就是空间的**维数**。

$\mathbb{R}^2$ 的维数是 $2$——你需要两个独立方向才能覆盖整个平面。$\mathbb{R}^3$ 的维数是 $3$——三个独立方向。一条直线（过原点）的维数是 $1$——一个方向就够了。

"三维空间有三个独立方向"这句话听起来像是废话——三维嘛，当然是三。但这里有一个微妙的逻辑反转：不是"空间是三维的所以需要三个独立方向"，而是"需要三个独立方向所以我们叫它三维"。维数不是先验的标签，而是空间自身结构的度量——你数一数需要多少个独立方向才能覆盖它，数出来几就是几维。

这个理解方式有什么好处呢？好处是巨大的。

---

## 一百维不是妄想

一旦你接受了"维数 = 基的个数"这个定义，维数就不再局限于你能想象的 $1$、$2$、$3$ 了。

次数不超过 $3$ 的多项式全体，构成一个向量空间。它的一组基是 $\{1, x, x^2, x^3\}$——四个"基向量"，线性无关，任何次数不超过 $3$ 的多项式都可以写成它们的线性组合。所以这个空间是**四维**的。

多项式不住在任何你能画出来的空间里，但它确实有四个独立方向。维数是代数结构决定的，不是几何想象决定的。

更进一步：$\mathbb{R}^{100}$——一百个实数组成的有序列表——构成一个一百维的向量空间。每一组一百个数就是一个"向量"，基有一百个元素。你画不出一百维的图，但这不影响你对它做线性代数的全套操作——加法、数乘、找基、算维数、一切照旧。

还有更猛的。连续函数构成的向量空间是**无穷维**的。为什么？因为你找不到一组有限的基能张成所有连续函数。$\{1, x, x^2, x^3, \ldots\}$ 这个无穷列表的线性组合只能覆盖多项式，连续函数比多项式多得多。你需要无穷多个"独立方向"才能覆盖它们。

无穷维空间，在十九世纪末二十世纪初被提出来的时候，很多人觉得这不过是数学家的抽象游戏——有什么用呢？你又不可能真的操作无穷多个坐标。

但一百年后，这些"抽象游戏"变成了最实用的工具。

当你在搜索引擎里输入一个关键词的时候，搜索引擎把整个互联网的每个网页表示成一个高维向量——维度可能是几千甚至几万。你的搜索词也被表示成一个向量。然后搜索引擎计算这些向量之间的"距离"或"夹角"，把最相关的结果排在前面。几千维？几万维？向量空间的公理不关心维数是多少——规则对三维成立，对三万维同样成立。

当机器学习算法处理一张图片的时候，一张 $100 \times 100$ 像素的灰度图可以看成一个一万维向量——每个像素的灰度值就是一个坐标。图像识别、人脸检测、风格迁移——这些听上去很酷炫的技术，底层都在高维向量空间里做线性代数运算。

皮亚诺在 1888 年写下那些公理的时候，GPU 还没有被发明，"机器学习"这个词还不存在。但他定义的那个抽象框架，恰好是一切的基础。

这就是数学的诡异之处——最"没用"的抽象，往往成为最有用的工具。不是因为数学家有预见性，而是因为他们剥离了所有具体细节之后，抓住的是最普遍的结构。而最普遍的结构，适用范围也最广。

## 下一个问题

好了，现在我们有了"空间"的概念。向量终于有了家——不管是箭头、多项式还是函数，只要满足那八条规矩，就住在一个向量空间里。

但光有空间还不够。

想想看，数学——尤其是应用数学——最关心的不是"东西在哪里"，而是"东西怎么变"。你不只是想知道一个向量住在哪个空间里，你更想知道：有没有一种操作，能把这个空间里的向量"搬运"到另一个空间去？

比如把二维平面上的点旋转九十度——每个点都被搬到了新的位置。比如把三维空间投影到一个平面上——每个三维向量都被"压扁"成了二维向量。

这些操作有一个共同特点：它们保持了线性结构。旋转之后，两个向量的和还是旋转后各自向量的和。投影之后，一个向量的两倍还是投影后那个向量的两倍。直线搬完还是直线，原点不动，比例关系不变。

这种"保持线性结构的搬运"，有一个名字：**线性变换**。

而线性变换跟我们的老朋友——矩阵——之间的关系，将会揭开本书第二幕最重要的秘密。
